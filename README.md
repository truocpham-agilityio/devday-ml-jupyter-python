# DevDay 2019 - Python Machine Learning Environment with Jupyter Notebook

## Getting Started

### Prerequisite

- Python 3

### Jupyter Notebook

Installing Jupyter with pip

```shell
python -m pip install jupyter
```

Running Jupyter Notebook

```shell
jupyter notebook --notebook-dir=notebooks
```
